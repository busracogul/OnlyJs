CSS Nedir?

CSS (Cascading Style Sheets), web sitelerinin görünümünü ve düzenini belirlemek için kullanılan bir stil dilidir. HTML ile birlikte çalışarak renkleri, yazı tiplerini, boşlukları ve diğer görsel öğeleri şekillendirir.
